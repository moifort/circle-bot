# polymarket-bot

## 📋 Requirements

- Bun
- Node 22
- [git-crytp](https://github.com/AGWA/git-crypt/blob/master/INSTALL.md)
- Ask for `.git-crypt-key` file and put it in the root of the project and run `git-crypt unlock .git-crypt-key`

## 🏗️ Setup

```bash
bun install
```

## Run

```bash
bun run index.ts
```
